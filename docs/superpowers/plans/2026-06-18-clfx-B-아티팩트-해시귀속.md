**Goal:** ~/.claude 기록(이벤트)이 참조한 **실제 파일**을 read-only로 해시/stat하는 포렌식 계층을 신설해 — ① 동일내용 복제(유출) 탐지, ④ 파일시스템 메타 vs transcript 증거 JOIN으로 행위주체(A/B) 왜곡 보정 — 을 결정적으로 산출하고 대시보드에 표시한다.

## 절대 불변식 (어기면 포렌식 실패 — 모든 Task에서 보증)
- **READ-ONLY FS**: `open(path,"rb")`(해시)·`os.stat`·`os.walk`(followlinks=False)만. **절대 쓰기/삭제/수정 금지**(open("w"), os.remove, os.rename, shutil.copy/move 등 금지).
- **완전성(최우선 — 사용자 강제약)**: **성능 위해 건너뛰거나 빼먹는 것 절대 금지.** tmp 전수 스캔은 모든 정규파일 해시(크기/개수 cap 금지·샘플링 금지·top-N 절단 금지). 큰 파일도 **스트리밍 해시로 전량**(skip 금지). 속도는 **병렬**로만 확보(coverage는 100%). 읽기 실패(권한/IO)는 **조용히 누락 금지** → `errors[]`에 {path,reason} 기록(정직한 보고).
- **무손실**: 이벤트/엔진/스키마 불변. 아티팩트는 추가 산출물. 참조 파일 부재 시 `missing` 카운트(②영역, 누락 아님).
- **결정성(I2/I4)**: 모든 출력 정렬(해시/경로/ts). 병렬 해시라도 그룹핑·정렬은 결정적. 테스트는 실제 머신 FS 비의존(tmp_path 픽스처 + os.name monkeypatch).
- **보안**: 시크릿 평문 노출 0(해시는 시크릿 아님). preview 표시 시 ‹secret› 마스킹 유지. 로컬만(외부전송 0).
- **견고성**: 파일별 try/except(권한거부·UNC 실패). os.walk followlinks=False(심볼릭링크 디렉터리 무한루프 차단). 비파일 target(명령/URL/[Pasted #N]/[Image]) skip. 정규파일만 해시(디바이스/소켓/fifo는 내용증거 아님 — errors 아님).

```python
import os
import re
import hashlib
from datetime import datetime, timezone

def _wsl_base_of(root):
    """root(\\wsl.localhost\Ubuntu\home\u\.claude) → \\wsl.localhost\Ubuntu. wsl 루트 아니면 None."""
    s = str(root)
    m = re.match(r"^(\\\\wsl\.localhost\\[^\\]+)", s) or re.match(r"^(\\\\wsl\$\\[^\\]+)", s)
    return m.group(1) if m else None

def resolve_candidates(target, root):
    """event.target(파일경로) → 분석 OS서 접근 가능한 실제 경로 후보. 비파일이면 []."""
    t = (target or "").strip()
    if not t or t.startswith("[") or "://" in t:        # [Pasted/Image], URL → 비파일
        return []
    on_nt = (os.name == "nt")
    m = re.match(r"^/mnt/([a-zA-Z])/(.*)$", t)            # WSL이 본 Windows 드라이브
    if m:
        drive, rest = m.group(1).upper(), m.group(2)
        return [f"{drive}:\\" + rest.replace("/", "\\")] if on_nt else [t]
    if re.match(r"^[a-zA-Z]:[\\/]", t):                  # Windows 절대경로
        if on_nt:
            return [t.replace("/", "\\")]
        return ["/mnt/" + t[0].lower() + "/" + t[2:].replace("\\", "/").lstrip("/")]
    if t.startswith("/"):                                # POSIX 절대경로(WSL distro fs)
        if on_nt:
            base = _wsl_base_of(root)
            return [base + t.replace("/", "\\")] if base else []
        return [t]
    return []                                            # 상대/비경로 skip

def resolve_existing(target, root):
    """후보 중 실제 존재하는 첫 경로(없으면 None). read-only stat만."""
    for c in resolve_candidates(target, root):
        try:
            if os.path.isfile(c):
                return c
        except OSError:
            continue
    return None
```

```python
def hash_file(path, chunk=1 << 16):
    """SHA-256 스트리밍(메모리 bound). read-only."""
    h = hashlib.sha256()
    with open(path, "rb") as f:
        for blk in iter(lambda: f.read(chunk), b""):
            h.update(blk)
    return h.hexdigest()

def _iso(epoch):
    return datetime.fromtimestamp(epoch, tz=timezone.utc).strftime("%Y-%m-%dT%H:%M:%S") + "Z"

def stat_file(path):
    st = os.stat(path)
    owner = None
    if os.name != "nt":
        try:
            import pwd
            owner = pwd.getpwuid(st.st_uid).pw_name
        except Exception:
            owner = None
    return {"size": st.st_size, "mtime": _iso(st.st_mtime), "owner": owner}
```
